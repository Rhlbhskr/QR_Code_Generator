# QR Forge

QR Forge is a simple, modern QR code generator that runs completely in your browser.
You can quickly turn any link or text into a QR code, optionally place a logo in the center, preview it instantly, and download it as a PNG image.

The goal of this project was to build a **clean, fast, and private tool** without needing a backend server or complicated setup.

---

## What you can do with it

* Generate QR codes for **links or any text**
* Add an **optional logo in the center**
* Adjust the **QR code size**
* Adjust the **logo size**
* **Drag and drop** a logo image
* Preview the QR code instantly
* **Download the QR code as a PNG**
* Everything runs **locally in your browser**

No data is sent anywhere — the entire tool works on the client side.

---

## How to use it

1. Open the site.
2. Enter a **URL or any text** in the input field.
3. (Optional) Upload a **logo image** to place in the center.
4. Adjust the **size settings** if you want.
5. Click **Generate QR Code**.
6. Download the finished QR code as a **PNG**.

That’s it.

---

## Project structure

```
qr-forge/
│
├── index.html
└── README.md
```

The entire project is built as a **single-page app**, so everything (HTML, CSS, and JavaScript) lives inside `index.html`.

---

## Technologies used

* HTML5
* CSS3
* Vanilla JavaScript
* QR generation powered by **qrcode.js**

No frameworks, no backend, and no external services required.

---

## Privacy

QR Forge was designed to be simple and private.

* No tracking
* No analytics
* No servers involved

Everything happens directly in your browser.
